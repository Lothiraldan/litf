What is LITF?

Language Independent Test Format is a new effort to have a language-independent test output and input format that is designed to be easy to produce and consume.

Easy to write and easy to read

LITF is designed to be easy both to producers and to consumers.

Unlock new usages

LITF is both an input and an output format which unlock new usages, like replaying CI failed tests locally or finely selecting which tests to run.

Clear definition

LITF is designed to be independent from any implementation, which ensures it is properly documented and not tied to any implicit language or test framework behavior.
